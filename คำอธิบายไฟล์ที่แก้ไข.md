### 1. `Application/Features/QORT09/Query/Master.cs`

**ประเภท:** แก้ไข (Fix Bug)

**หน้าที่ของไฟล์:**
ไฟล์นี้เป็น MediatR Query Handler ทำหน้าที่ดึงข้อมูล Dropdown ทั้งหมดที่ใช้ในหน้าจอ QORT09, QORT10, QORT11
ประกอบด้วย 5 dropdown:
- `DDLProType` — ประเภทโปรโมชั่น (จากตาราง `qo_pro_type`)
- `DDLCurrency` — สกุลเงิน (จากตาราง `db_currency`)
- `DDLStatus` — สถานะเอกสาร (N/W/A/R/C)
- `DDLPromotion` — รายการโปรโมชั่น (จากตาราง `qo_pro_master`)
- `DDLGoodsService` — สินค้าและบริการ (รวมจาก `in_goods` + `iv_service_item`)

**สิ่งที่แก้ไข:**
ค่าที่กำหนดให้ `DropDownDto` ถูกใส่สลับกันระหว่าง `Value` (string) และ `Value2` (int)

| Method | Property | เดิม | แก้เป็น |
|---|---|---|---|
| `GetProType()` | `Value` | `t.ProTypeId` (int ❌) | `t.ProTypeCode` (string ✓) |
| `GetPromotion()` | `Value` | `p.ProId` (int ❌) | `p.ProCode` (string ✓) |
| `GetPromotion()` | `Value2` | `p.ProCode` (string ❌) | `p.ProId` (int ✓) |
| `GetGoodsService()` | `Value` | `g.ItemId` (int ❌) | `g.ItemCode` (string ✓) |
| `GetGoodsService()` | `Value2` | `g.ItemCode` (string ❌) | `g.ItemId` (int ✓) |
| `GetGoodsService()` (service) | `Value` | `s.ItemId` (int ❌) | `s.ItemCode` (string ✓) |
| `GetGoodsService()` (service) | `Value2` | `s.ItemCode` (string ❌) | `s.ItemId` (int ✓) |

### 2. `Web/Web.csproj`

**ประเภท:** แก้ไข (Configuration)

**หน้าที่ของไฟล์:**
ไฟล์ Project Definition ของ layer **Web** (ASP.NET Core API + Angular host)
กำหนด dependencies, target framework, และ package ที่ใช้ในโปรเจกต์นี้

**สิ่งที่แก้ไข:**

```xml
<!-- เดิม -->
<TargetFramework>net8.0</TargetFramework>

<!-- แก้เป็น -->
<TargetFramework>net10.0</TargetFramework>
```

**เหตุผล:** เครื่องติดตั้ง .NET 10 แต่ไฟล์กำหนด net8.0 ทำให้รัน app ไม่ได้

### 6. `Persistense/Persistense.csproj`

**ประเภท:** แก้ไข (Configuration)

**หน้าที่ของไฟล์:**
ไฟล์ Project Definition ของ layer **Persistence**
เป็น layer ที่จัดการฐานข้อมูลทั้งหมด ได้แก่:
- EF Core DbContext (`CleanDbContext`)
- Dapper (raw SQL)

- Custom Configuration Provider (ดึง config จาก DB ตอน startup)
ใช้ package: Npgsql, EF Core, Dapper, AspNetCore.Identity
