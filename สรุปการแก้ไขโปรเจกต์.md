
# สรุปการแก้ไขและตั้งค่าโปรเจกต์ ssclean

> วันที่: 2 มิถุนายน 2026
> โปรเจกต์: ssclean (Clean Architecture + Angular)

---

## โครงสร้างโปรเจกต์โดยรวม

โปรเจกต์นี้เป็นระบบ **Quote Order (QO)** ของฝ่ายการเงิน (FINC) ใช้สถาปัตยกรรม Clean Architecture แบ่งเป็น 5 Layer:

```
Domain → Application → Infrastructure / Persistense → Web (API + Angular)
```

| Layer          | โฟลเดอร์    | หน้าที่                                        |
| -------------- | ------------------- | ----------------------------------------------------- |
| Domain         | `Domain/`         | Entity, Enum, Event, Exception — core ของระบบ |
| Application    | `Application/`    | Business logic, CQRS (Command/Query), Interface       |
| Infrastructure | `Infrastructure/` | External service เช่น Email, CurrentUser          |
| Persistence    | `Persistense/`    | DbContext (EF Core + Dapper), Identity, Config        |
| Presentation   | `Web/`            | ASP.NET Core API + Angular Frontend                   |

### ปัญหาที่ 1 — Build Error: Type mismatch ใน QORT09/Query/Master.cs

**ไฟล์ที่แก้:** `Application/Features/QORT09/Query/Master.cs`

**สาเหตุ:**
`DropDownDto` กำหนด type ดังนี้:

```csharp
public string Value  { get; set; }  // รับ string
public int    Value2 { get; set; }  // รับ int
```

แต่โค้ดใน `Master.cs` กำหนดค่าสลับกัน ทำให้เกิด error `CS0029: Cannot implicitly convert type`

**Error ที่เกิด (7 จุด):**

```
QORT09/Query/Master.cs(53)  : error CS0029: Cannot implicitly convert type 'int' to 'string'
QORT09/Query/Master.cs(82)  : error CS0029: Cannot implicitly convert type 'int' to 'string'
QORT09/Query/Master.cs(83)  : error CS0029: Cannot implicitly convert type 'string' to 'int'
QORT09/Query/Master.cs(93)  : error CS0029: Cannot implicitly convert type 'int' to 'string'
QORT09/Query/Master.cs(94)  : error CS0029: Cannot implicitly convert type 'string' to 'int'
QORT09/Query/Master.cs(103) : error CS0029: Cannot implicitly convert type 'int' to 'string'
QORT09/Query/Master.cs(104) : error CS0029: Cannot implicitly convert type 'string' to 'int'
```

**สิ่งที่แก้ไข:**

| Method                         | เดิม (ผิด)                | แก้เป็น (ถูก)            |
| ------------------------------ | -------------------------------- | ---------------------------------- |
| `GetProType()`               | `Value = t.ProTypeId` (int)    | `Value = t.ProTypeCode` (string) |
| `GetPromotion()`             | `Value = p.ProId` (int)        | `Value = p.ProCode` (string)     |
| `GetPromotion()`             | `Value2 = p.ProCode` (string)  | `Value2 = p.ProId` (int)         |
| `GetGoodsService()` goods    | `Value = g.ItemId` (int)       | `Value = g.ItemCode` (string)    |
| `GetGoodsService()` goods    | `Value2 = g.ItemCode` (string) | `Value2 = g.ItemId` (int)        |
| `GetGoodsService()` services | `Value = s.ItemId` (int)       | `Value = s.ItemCode` (string)    |
| `GetGoodsService()` services | `Value2 = s.ItemCode` (string) | `Value2 = s.ItemId` (int)        |

**โค้ดหลังแก้ไข (ส่วนที่เปลี่ยน):**

```csharp
// GetProType — ใช้ ProTypeCode (string) แทน ProTypeId (int)
select new DropDownDto
{
    Value = t.ProTypeCode,                          // string ✓
    Text  = t.ProTypeCode + " - " + t.ProTypeNameTh,
}

// GetPromotion — สลับ Value ↔ Value2
select new DropDownDto
{
    Value  = p.ProCode,   // string ✓
    Value2 = p.ProId,     // int    ✓
    Text   = p.ProCode + " - " + p.ProDescTh,
}

// GetGoodsService (goods) — สลับ Value ↔ Value2
select new DropDownDto
{
    Value  = g.ItemCode,  // string ✓
    Value2 = g.ItemId,    // int    ✓
    Text   = g.ItemCode + " - " + g.ItemName,
    Text2  = "F",
}

// GetGoodsService (services) — สลับ Value ↔ Value2
select new DropDownDto
{
    Value  = s.ItemCode,  // string ✓
    Value2 = s.ItemId,    // int    ✓
    Text   = s.ItemCode + " - " + s.ItemName,
    Text2  = "S",
}
```

### ปัญหาที่ 2 — Runtime Error: .NET version ไม่ตรง

**สาเหตุ:**
เครื่องติดตั้ง **.NET 10** แต่โปรเจกต์กำหนด `<TargetFramework>net8.0</TargetFramework>` ทำให้รัน app ไม่ได้

**Error ที่เกิด:**

```
You must install or update .NET to run this application.
Framework: 'Microsoft.NETCore.App', version '8.0.0' (arm64)
The following frameworks were found:
  10.0.8 at [/usr/local/share/dotnet/shared/Microsoft.NETCore.App]
```

**ไฟล์ที่แก้ไข (5 ไฟล์):**

| ไฟล์                                 | เดิม   | ใหม่    |
| ---------------------------------------- | ---------- | ----------- |
| `Web/Web.csproj`                       | `net8.0` | `net10.0` |
| `Application/Application.csproj`       | `net8.0` | `net10.0` |
| `Domain/Domain.csproj`                 | `net8.0` | `net10.0` |
| `Infrastructure/Infrastructure.csproj` | `net8.0` | `net10.0` |
| `Persistense/Persistense.csproj`       | `net8.0` | `net10.0` |

**วิธีแก้ (command ที่ใช้):**

```bash
sed -i '' 's/<TargetFramework>net8.0<\/TargetFramework>/<TargetFramework>net10.0<\/TargetFramework>/g' <ชื่อไฟล์>
```

### ปัญหาที่ 3 — Database Connection Timeout (ยังไม่ได้แก้ — รอ VPN)

**Error ที่เกิด:**

```
Npgsql.NpgsqlException: Failed to connect to 172.16.0.49:5434
System.TimeoutException: Timeout during connection attempt
```

**สาเหตุ:**
DB Server อยู่ใน private network ขององค์กร (IP: `172.16.0.49:5434`)
เครื่องยังไม่ได้เชื่อมต่อ VPN

**วิธีแก้:**

1. เชื่อมต่อ VPN ขององค์กร
2. รัน backend ใหม่: `dotnet run --environment Development`

### ปัญหาที่ 4 — Frontend หน้าดำ (ผลมาจากปัญหาที่ 3)

**Error ใน Browser Console:**

```
Failed to load resource: net::ERR_CONNECTION_REFUSED
→ https://localhost:5001/api/configuration

TypeError: Failed to fetch
→ ConfigurationService.loadConfig (configuration.service.ts:16)
```

**สาเหตุ:**
Angular ดึง config จาก Backend (`/api/configuration`) **ก่อนโหลดหน้าแรก**
ถ้า Backend ไม่ทำงาน → app โหลดไม่ได้ → หน้าดำ

**วิธีแก้:** ต้องรัน Backend ให้สำเร็จก่อน (แก้ปัญหา VPN ข้างบน)

## วิธีรัน Project (ถูกต้อง)

### สิ่งที่ต้องมี

- .NET SDK 10+
- Node.js + npm
- เชื่อมต่อ VPN ขององค์กรแล้ว

### ขั้นตอน

**Terminal 1 — รัน Backend:**

```bash
cd "/path/to/ssclean/Web"
dotnet run --environment Development
# รอจนขึ้น: Now listening on: https://localhost:5001
```

**Terminal 2 — รัน Frontend:**

```bash
cd "/path/to/ssclean/Web/ClientApp"
npm install        # ครั้งแรกเท่านั้น
npm start
# รอจนขึ้น: Compiled successfully
```

**เปิด Browser:**

```
http://localhost:4200
```

### ลำดับสำคัญ

```
VPN → dotnet run → npm start → เปิด localhost:4200
```

> ⚠️ ต้องรัน Backend ก่อนเสมอ เพราะ Angular เรียก API ตั้งแต่ตอน startup
